Qual a diferença entre a linguagem de programação client side e server side:

client side:

As linguagens client-side são linguagens onde apenas o seu NAVEGADOR vai entender (Ex. Javascript)

Server side:

As linguagens server-side são linguagens que o SERVIDOR entende. Isso quer dizer que vai escrever um código onde o servidor vai processá-lo e então vai mandar para o seu navegador a resposta (Ex. PHP, JAVA)

diferença entre os dois:

client side:

  • são linguagens onde apenas o seu NAVEGADOR vai entender.
  • Quem vai processar essa linguagem não é o servidor, mas o seu browser.

server side:

  • É responsável por processar as solicitações feitas pelos usuários e fornecer as respostas adequadas.
  • o server-side lida com a lógica de negócios, acesso a banco de dados, autenticação, entre outras funcionalidades.
